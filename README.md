# Stick Fight Arena

A browser stick-figure fighting game with combos, parries, and a 9-hit ultimate.

**Play it:** open the GitHub Pages link for this repo.

## Controls

| Action | Player 1 | Player 2 |
| --- | --- | --- |
| Move | A / D | ← / → |
| Jump (double) | W | ↑ |
| Block | S | ↓ |
| Light attack | F | K |
| Heavy attack | G | L |
| Dash | Q | ; |
| Ultimate | R | Enter |

Chain light attacks for combos. Block in the first 6 frames of an attack to perfect-parry. Build the ultimate meter, then unleash a 9-hit cinematic finisher.

First fighter to win 2 rounds takes the match.
